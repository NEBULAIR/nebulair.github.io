A fully automated aquaponics system!

Aquaponics systems are closed/semi-closed eco systems that grow fish and vegetables biologically. Nebulair has an open source Github Repository with all the information about how to make your own automated aquaponics system, including the code and order list for the hardware.

Why Nebulair?

Open Source

You can find all the information to make your own automated aquaponics system in our repo

Internet of things

We connect all the sensing data, pumping and lighting to the internet so you can controll your aquaponics system from anywhere.

Api and code

We have written all the code so you can get started right away. A awesome APi to optimize your system is coming soon!

Be Healthy

We started this project to try and make a difference. Aquaponics means no fertilizers and 90% reduction in water usage compared to traditional farming methods!
